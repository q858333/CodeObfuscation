# codeObfuscation
创建.sh文件
首先需要一个文本编辑工具，可以下载vim编辑器（下载地址http://code.google.com/p/macvim/），也可以用系统自带的文本编辑器，在里面输入代码。
如下：
#! /bin/bash  
echo haha  

然后另存文件为**.sh，打开shell进入刚才保存的目录，执行chmod +x myscript.sh，到这儿为止，就建立好了。
直接把**.sh拖到shell里面回车就会看到haha的结果。
